Описание программы.

Данная программа была разработана на языке Python (v.3.6.9) на базе фрейморка Django (v. 3.2.6).
В ней была реализована автоматизация документооборота. Также для создания графического интерфейса былы использованы HTML5/CSS3 и библиотека Bootstrap 4.

Входными данными являются:

  • данные о сотруднике (ФИО);
  • данные о адресе (страна, город, адрес);
  • данные о телефоне (телефон, примечание);
  • данные о должности (отдел, должность);
Выходными данными являются:
  • данные о сотруднике (ФИО);
  • данные о адресе (страна, город, адрес);
  • данные о телефоне (телефон, примечание);
  • данные о должности (отдел, должность);
Требования к временным характеристикам:
  • программный продукт должен быстро реагировать на действия пользователя;
  • задержка при загрузке не должна превышать 10 сек;
  • исключить зависания и остальные не предвиденные ошибки программного продукта;
  • задержка при создании отчетов не должна превышать 10 сек;
  • занесение записи в базу должно происходить не более чем за 5 сек.
Требования к пользовательскому интерфейсу:
  • наличие понятного интерфейса;
  • наличие цветовой палитры не раздражающего цвета;
  • возможность работать с программой не подготовленному пользователю;
  • не нагромождение таблиц в одном окне, а вывод необходимых по запросу пользователя.
Требования к надёжности: Программный продукт должен быть надёжно организован в области хранения данных. Для того чтобы обеспечивать наилучшую надёжность при работе системы необходимо следить за следующими ограничениями:
  • в поля, предназначенные для записи текста нельзя вводить числовые значения и наоборот, если же будут введены неверные данные, тогда должно появиться сообщение об ошибке;
  • разграничить доступ к данным;
  • запретить некорректных символов в программу.
Для поддержания целостности БД необходимо:
  • - использование первичного ключа;
  • - приведение к нормальной форме.

Условия эксплуатации: Для эксплуатации одного экземпляра программы необходим минимум один человек (Оператор ПК), который будет запускать программу и вносить данные в неё. Для этого необходимо умение владения персональным компьютером на уровне пользователя.
